# Super Trunfo – Países (C)

Implementação do jogo **Super Trunfo – Países** em linguagem C, com foco em lógica de decisão, comparação de atributos e interação via terminal.

---

## Visão Geral

O projeto simula a lógica principal do jogo Super Trunfo, permitindo o cadastro de cartas e a comparação de seus atributos para determinar a carta vencedora conforme regras específicas do jogo.

A aplicação é executada via terminal e utiliza estruturas condicionais para tomada de decisão.

---

## Funcionalidades

- Cadastro de cartas com múltiplos atributos
- Comparação entre cartas baseada em critérios definidos
- Regras específicas por atributo (maior ou menor valor vence)
- Menu interativo para seleção de comparações
- Tratamento de empates
- Comparação de múltiplos atributos simultaneamente

---

## Estrutura das Cartas

Cada carta contém:

- Estado  
- Código da carta  
- Nome da cidade  
- População  
- Área  
- PIB  
- Pontos turísticos  
- Densidade populacional (calculada)
